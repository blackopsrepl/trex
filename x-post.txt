I built trex -- a tmux session manager in Rust that shows live CPU, memory, and health scores for every session. It tracks running AI coding agents (Claude, OpenCode), has fuzzy search, pane previews, git status inline, and sparkline charts. Basically htop for tmux.

Built for Omarchy (Basecamp's Linux desktop), works anywhere.
